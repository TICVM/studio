# PessoasEmpresa - Carômetro Corporativo

Este é um sistema de gestão visual de colaboradores (Carômetro) desenvolvido com Next.js, Tailwind CSS e Firebase.

## 🚀 Como publicar no GitHub

Para enviar este projeto para o seu GitHub, siga os passos abaixo no terminal:

1. **Inicie o repositório local:**
   ```bash
   git init
   ```

2. **Adicione os arquivos:**
   ```bash
   git add .
   ```

3. **Crie o primeiro commit:**
   ```bash
   git commit -m "Initial commit: Carômetro completo com painel administrativo"
   ```

4. **Conecte ao seu GitHub:**
   - Crie um novo repositório no [GitHub](https://github.com/new).
   - Copie a URL do repositório (algo como `https://github.com/seu-usuario/seu-repositorio.git`).
   - No terminal, execute:
     ```bash
     git remote add origin https://github.com/seu-usuario/seu-repositorio.git
     git branch -M main
     git push -u origin main
     ```

## ✨ Funcionalidades

- **Painel Público:** Visualização de todos os colaboradores ativos com filtros por Nome, Cargo, Setor e Unidade.
- **Área Administrativa:** Gestão completa de Funcionários e Setores.
- **Estúdio Visual:** Personalização total de cores, logos e layout dos cards sem mexer no código.
- **Edição em Massa:** Altere a unidade de vários funcionários de uma só vez.
- **Importação:** Suporte para importação de dados via planilhas Excel.

## 🛠️ Tecnologias Utilizadas

- **Framework:** [Next.js 15](https://nextjs.org/)
- **Estilização:** [Tailwind CSS](https://tailwindcss.com/) + [Shadcn/UI](https://ui.shadcn.com/)
- **Banco de Dados & Auth:** [Firebase](https://firebase.google.com/)
- **Ícones:** [Lucide React](https://lucide.dev/)
